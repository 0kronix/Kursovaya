# FBT to SVG Converter

Программа для конвертации функциональных блоков IEC 61499 из XML (.fbt) в SVG.

## Быстрая сборка (Windows)

1. Скачайте проект
2. Дважды кликните на `build.bat`
3. Исполняемый файл `fb_to_svg_converter.exe` появится в папке проекта

## Использование

1. Поместите необходимые для конвертации файлы в формате `.fbt` в папку `resources/fbt/`
2. Запустите испольняемый файл
3. Получнные файлы в формате `.svg` будут находиться в папке `resources/svg/`
